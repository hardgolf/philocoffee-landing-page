# Prof Doctor of Philocoffee Landing Page

A simple one-page Vite landing page for a cafe and coffee learning concept in Ekkamai, Bangkok.

## Features

- Hero section with a small Three.js coffee cup animation
- About, Services, Courses, and Contact sections
- Responsive layout for desktop and mobile
- Lightweight Vite setup

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
